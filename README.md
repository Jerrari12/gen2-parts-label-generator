# GEN2 Label Generator · Jerrari3D

A browser-based tool for creating custom labels for GEN2 Modular Storage System parts drawers. No install, no account, no fuss, open it and go.

🔗 **[Use it live →](https://jerrari12.github.io/gen2-parts-label-generator/)**

---

## What it does

Generates print-ready labels sized to fit all GEN2 parts drawers. One universal label size works with all GEN2 Parts Drawer sizes
---

## How to use it

1. Enter your label text
2. Adjust style options as needed
3. Export and print

That's it.

---

## Run it locally

No server needed. Just download `index.html` and open it in any modern browser.

---

## Compatible drawers

Designed for all GEN2 Parts Drawers.


Part of the [GEN2 Modular Storage System](https://jerrari3d.com) by Jerrari3D.
